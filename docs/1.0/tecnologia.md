# Tecnologia

!> **Atenção** Você está vendo uma documentação antiga. A versão v1.0.0 foi lançada em Dezembro de 2013.

- PHP 5.3+
- HTML5
- CSS3
- Doctrine PHP 2.4
- jQuery 1.9+
- Twitter Bootstrap 3
- Highcharts

Testado no seguintes bancos:

- PostgreSQL
- MySQL
- MS SQL
